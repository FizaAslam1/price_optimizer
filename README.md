

# 🎯 PriceOptimizer AI

**AI-Powered Dynamic Pricing Engine — Maximize Profit with Demand Prediction**

---

## 🚀 What It Does

Predicts how many units will sell at **any price point**, then finds the price that **maximizes total profit**.
Profit = (Price - Cost) × Predicted Demand


## ✨ Features

| Feature | Description |
|---------|-------------|
| 📈 **Demand Prediction** | XGBoost model predicts units sold at any price |
| 🎯 **Profit Optimization** | Tests 80+ price points to find maximum profit |
| 🔄 **What-If Simulator** | See impact of competitor price changes instantly |
| 📊 **Interactive Charts** | Demand curve + Profit curve with optimal point |
| 💡 **Strategy Suggestions** | AI recommends: Raise, Lower, or Hold price |

## 🎮 How To Use

1. Adjust **vehicle specs** in sidebar (year, mileage, horsepower...)
2. Set **cost price** (what you paid)
3. Set **competitor average price**
4. See **optimal selling price** instantly
5. Try **What-If** tab to simulate market changes
